# admin-core-nexus

Kullanıcı ve ürün yönetim süreçlerini merkezileştiren, gerçek zamanlı veri takibi sağlayan kapsamlı admin paneli.

## Teknoloji

| | |
|---|---|
| Mimari | react-vite |
| Frontend | React (Vite), Tailwind CSS, Headless UI, Recharts |
| Backend | Node.js (Express) or Python (FastAPI) |
| Veritabani | PostgreSQL |
| Deployment | Docker, AWS or Vercel |

## Kurulum

```bash
cd backend
npm install
npm run dev

# Baska terminal
cd frontend
npm install
npm run dev
```
