# Kalite Kontrol Listesi
## HTML Validasyonu
- [ ] HTML kodlarının W3C standartlarına uygun olması
- [ ] HTML etiketlerinin doğru bir şekilde kullanılmış olması

## CSS Uyumluluğu
- [ ] CSS kodlarının W3C standartlarına uygun olması
- [ ] CSS selectors'un doğru bir şekilde kullanılmış olması

## JS Hataları
- [ ] JS kodlarının hata vermeden çalışması
- [ ] JS fonksiyonlarının doğru bir şekilde tanımlanmış olması

## Responsive
- [ ] Sayfaların mobil, tablet ve masaüstü cihazlarda doğru bir şekilde görünmesi
- [ ] Sayfaların farklı ekran boyutlarında doğru bir şekilde uyarlanması

## SEO
- [ ] Meta etiketlerinin doğru bir şekilde kullanılmış olması
- [ ] Heading hierarchy'nin doğru bir şekilde kullanılmış olması
- [ ] Alt text'in doğru bir şekilde kullanılmış olması

## Güvenlik
- [ ] XSS 공격larına karşı koruma
- [ ] CSRF saldırılarına karşı koruma

## Performans
- [ ] Sayfaların 3 saniyeden az sürede yüklenmesi
- [ ] Resim boyutlarının optimize edilmiş olması
- [ ] JS ve CSS dosyalarının minification yapılmış olması

## Erişilebilirlik
- [ ] Sayfaların WCAG 2.1 AA standartlarına uygun olması
- [ ] Renk kontrastının yeterli olması
